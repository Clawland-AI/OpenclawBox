# ClawBox — Product Fact Sheet

## One-Liner

ClawBox is a plug-and-play personal AI appliance that routes across 38+ models to save up to 90% on AI costs.

## Core Value Propositions

1. **Cheaper**: Smart routing saves up to 90% on AI inference costs
2. **Reliable**: Auto-failover across providers — zero downtime
3. **Governed**: Fleet management, OTA updates, audit trails, rate limiting

## Hardware Highlights

- Rockchip RK3588/RK3588S — 8-core ARM, 6 TOPS NPU
- 7.9" IPS bar touchscreen (1280x400), capacitive touch
- WiFi 6, Bluetooth 5.2, Gigabit Ethernet
- Compact: 140 x 100 x 35 mm, ~280-320g
- Matte black injection-molded enclosure

## Software Highlights

- OpenAI-compatible API (`/v1/chat/completions`)
- 38+ models across 7 providers via ClawRouter
- 4 routing profiles: eco / auto / premium / free
- Fleet management with web dashboard
- Metadata-only audit logging (no prompt storage by default)
- 100% open source (Apache-2.0)
